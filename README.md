# Stroke-prediction-

##### Autoria: 
Projeto desenvolvido em conjunto por:
- Naira Gama
- Nadiellen de Melo
  
### Objetivo: 
Aplicação dos modelos de Regressão Logística e Random Forest para prever a probabilidade de ocorrência de Acidente Vascular Cerebral (AVC). 

### Tecnologias utilizadas:
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn

### Etapas:
- Verificação dos Dados
- Análise Exploratória dos Dados (EDA)
- Limpeza dos Dados
- Aplicação dos algoritmos com dados desbalanceados
- Aplicação de técnicas de balanceamento
- Aplicação dos algoritmos com dados balanceados
- Avaliação final
